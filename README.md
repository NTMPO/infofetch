# Install for arch linux(AUR) - SOON:
  ```bash
yay -S infofetch
```
# Manual build(other distributions):
``` bash
git clone https://github.com/NTMPO/infofetch
cd infofetch 
make
sudo make install
```
# Use:
```bash
infofetch
```
